## Descrição
Este projeto foi desenvolvido para criar um espaço online onde pessoas possam desabafar e encontrar ajuda emocional.

## ODS relacionado
ODS 3 - Saúde e Bem-Estar: É relacionado por saúde mental e bem-estar das pessoas.
ODS 8 - Trabalho Decente e Crescimento Econômico: É relacionado porque uma boa saúde mental faz que a pessoa trabalhe bem.
